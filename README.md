# Chess Demo

### [▶ Play Now](https://minyic7.github.io/claude-hub-demo-chess/)

A fully playable chess game in a single HTML file — zero dependencies, zero build step.

Built entirely by [Claude Code](https://claude.ai).

---

> **Best experienced via the [live demo](https://minyic7.github.io/claude-hub-demo-chess/)** — no install required.

## Features

- **Complete chess rules** — all standard piece movements
- **Special moves** — castling, en passant, pawn promotion with piece selection UI
- **Check, checkmate, and stalemate detection**
- **Two-player local mode** — human vs human
- **AI opponent** — minimax search with 3–4 move lookahead
- **Play as White or Black** against the AI
- **Captured pieces display**
- **Game status display** — whose turn, check, checkmate, stalemate
- **New game button**
- **Unicode chess pieces** — no external assets needed
- **Responsive design**

## Tech Stack

- Single `index.html` file — all HTML, CSS, and JavaScript in one file
- Zero external dependencies
- Zero build step
- Deployed via GitHub Pages

## Run Locally

Open `index.html` in any browser:

```bash
# Option 1: open directly
open index.html

# Option 2: use a local server
python3 -m http.server
```

## Deploy

Push to the `main` branch — GitHub Pages auto-deploys.
